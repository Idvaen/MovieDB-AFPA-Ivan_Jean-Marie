Créateurs: Ivan Denysov, Jean-Marie Gassiès.
Création d'une application web responsive basée sur The Movie Data Base.
Elle vous permet d'accéder aux informations comme l'année, le synopsis ou la liste des acteurs pour chaque film.
elle est composée dans sa barre de navigation de 6 items vous permettant d'accéder aux divers informations:
-acceuil
-liste description: qui vous permettra de voir la liste des films puis en cliquant sur l'image d'accéder à plus d'informations.
-actuellement en salle: vous permettra d'accéder aux films en salle.
-les plus populaires: vous permettra de voir les films déjà sortis les plus populaires
-les mieux notés:vous permettra de voir les films déjà sortis ayant les meilleurs notes.
-bientot en salle: vous indique les films qui vont bientot sortir.
